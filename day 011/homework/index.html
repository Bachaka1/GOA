HTML • CSS • JavaScript

html, css და JavaScript

ვებ დეველოპმენტის სამი ძირითადი ტექნოლოგია

html

ჰიპერტექსტური აღწერის ენა (HTML - HyperText Markup Language) — ტიმ ბერნერს-ლის მიერ 1993 წელს შექმნილი აღწერის ენა, რომელიც განკუთვნილია ვებ-გვერდების და საიტების შესაქმნელად და ინფორმაციის გასავრცელებლად ინტერნეტის საშუალებით.

css

CSS არის სტილების ენა, რომელიც გამოიყენება HTML-ის ელემენტების დიზაინისთვის — ფერები, ზომები, განლაგება, ანიმაციები და ა.შ.

JavaScript

JavaScript არის მაღალი დონის პროგრამირების ენა, რომელიც გამოიყენება ვებგვერდებზე ინტერაქტიული და დინამიკური ფუნქციონალის შესაქმნელად.
